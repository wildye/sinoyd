# evaluate

## 配置 src 目录
```
├── App.vue                  // APP入口文件
├── api                      // 接口调用工具
│   └── index.js
├── components               // 组件
├── store                    // 状态管理
│   └── index.js

├── router                   // 路由配置文件夹
│   └── index.js                // 路由配置文件
├── view                     // 页面
│   ├── index.vue                // 问卷列表页、主页
│   ├── login.vue                // 登录页
│   ├── summary.vue              // 问卷汇总页
│   └── survey.vue               // 问卷答卷页
├── App.vue                  // APP入口文件
└── main.js                  // 项目配置文件
```

## 配置 static 目录
```
├── css             // 放一些第三方的样式文件
├── fonts           // 放字体图标文件
├── image           // 放图片文件，如果是复杂项目，可以在这里面再分门别类
└── js              // 放一些第三方的JS文件，如 jquery
```
